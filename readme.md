
# 网易云音乐移动端页面仿制
注：这是一个非官方的网易云音乐手机端播放页面demo.  

## 预览链接 
(请使用移动端设备预览）
https://majianyang666.github.io/music-demo-201708/home.html

## 页面的构成 
1. 主页面
  - 推荐音乐
  - 热歌榜
  - 搜索栏及搜索功能
2. 播放页面
  - 音乐播放
  - 歌词滚动

## 技术 
### 数据库
LeanCloud + 七牛云
### 页面 （Page）
原生JS + jQuery + requireJS/gulp/webpack
